# 🌐 Networking Basics

## What is Networking?
Networking is the process of connecting computers and devices to share data.

## Types of Networks
- LAN (Local Area Network) → Small area (home, college)
- WAN (Wide Area Network) → Large area (Internet)

## IP Address
An IP address is a unique identifier for a device on a network.
Example: 192.168.1.1

## MAC Address
A physical address assigned to a device (unique for each device).

## DNS (Domain Name System)
Converts domain names into IP addresses.
Example: google.com → 142.x.x.x

## Ports
Ports are communication endpoints.
Examples:
- 80 → HTTP
- 443 → HTTPS
- 22 → SSH

## Protocols
Rules for communication:
- HTTP / HTTPS → Web
- FTP → File transfer
- TCP → Reliable connection
- UDP → Fast but less reliable
